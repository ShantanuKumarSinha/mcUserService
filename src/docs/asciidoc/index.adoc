== Introduction

This is the documentation for the `mcUserService` project.

== API Endpoints

=== User Authentication
* **POST** `/users/authenticate`
  - Request Body:
    ```json
    {
      "email": "test@example.com",
      "password": "password123"
    }
    ```
  - Response:
    ```json
    {
      "token": "abc123"
    }
    ```

== Database Schema

[cols="2,3", options="header"]
|===
| Column | Description
| id | Primary key
| email | User email
| password | Encrypted password
|===
